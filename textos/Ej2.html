Este es un 
            ejemplo     de      texto
            preformateado.
            La marca pre        preserva      espacios      en   blanco
            y saltos de línea

La marca pre es buena para mostrar código de programación: 

            for i = 1 to 10
                print i 
            next i

También se puede utilizar la marca code

for i = 1 to 10 print i next i

Pero code no respeta espacios ni saltos de línea
